# ATOM DTU CAT1

## Overview

Contains M5Stack-**ATOM_DTU_CAT1** series related case programs. The hardware is based on the SIMA7680C series of modules to realize the CAT1 communication function.

## Libraries

- [ArduinoModbus](https://github.com/m5stack/ArduinoModbus)
- [Arduino485](https://github.com/m5stack/ArduinoRS485)
- [TinyGSM](https://github.com/vshymanskyy/TinyGSM)
- [PubSubClient](https://github.com/knolleary/pubsubclient.git)
